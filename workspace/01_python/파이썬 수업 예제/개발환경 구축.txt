[1. 기본 파이썬 환경]
- https://python.org
- 메뉴의 Downloads 클릭
- 원하는 버전을 다운로드 받는다.
- 설치 시작시 "Add Python x.xx to PATH"
  를 체크해준다.
- 설치 완료 후 확인 : cmd 실행 후 
  python -V
